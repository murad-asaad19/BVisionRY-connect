Key invariants:
• Every intro request gates on a moderation pass before delivery.
• Daily matches refresh at user-local 04:00; eligibility uses the 6-week activity filter.
• LLM unavailability degrades to heuristics-only ranker + plain-form onboarding — never blocks the user.
• Public profile pages reveal a strict subset; verified badges hidden, verified-Investor pages default OFF.
